---

## 简介

根据 `task_id` 轮询任务状态与结果。建议每 **5 秒**查询一次，视频通常 **30~120 秒**内完成。

状态流转：`queued` → `in_progress` → `completed` / `failed`

## 认证

<ParamField header="Authorization" type="string" required>
  Bearer Token，如 `Bearer sk-xxxxxxxxxx`
</ParamField>

## 路径参数

<ParamField path="task_id" type="string" required>
  创建任务返回的 `id` 或 `task_id`
</ParamField>

## 请求示例

```bash
curl -X GET "https://api.tennda.ai/v1/video/generations/ut-abc123def456" \
  -H "Authorization: Bearer sk-xxxxxxxxxx"
```

## 状态说明

| status | 说明 |
| --- | --- |
| `queued` | 排队中 |
| `in_progress` | 生成中，查看 `progress` |
| `completed` | 成功，读取 `video_url` 或 `url` |
| `failed` | 失败，查看 `error.message`；审核类失败可 [查询审核拦截原因](/cn/api-reference/endpoint/seedance-2.0/get-moderation-result) |

## 响应示例

**生成中：**

```json
{
  "id": "ut-abc123def456",
  "task_id": "ut-abc123def456",
  "object": "video",
  "model": "seedance-2-0",
  "status": "in_progress",
  "progress": 50,
  "created_at": 1712563200
}
```

**成功：**

```json
{
  "id": "ut-abc123def456",
  "task_id": "ut-abc123def456",
  "status": "completed",
  "progress": 100,
  "video_url": "https://uptoken.cc/v1/media/proxy?...",
  "url": "https://uptoken.cc/v1/media/proxy?...",
  "created_at": 1712563200,
  "completed_at": 1712563320,
  "metadata": {
    "status": "succeeded",
    "usage": { "total_tokens": 97605 }
  }
}
```

**失败：**

```json
{
  "id": "ut-abc123def456",
  "status": "failed",
  "progress": 100,
  "error": {
    "message": "Content moderation failed",
    "code": "failed"
  }
}
```
