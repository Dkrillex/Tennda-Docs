</ParamField>

## 路径参数

<ParamField path="virtual_id" type="string" required>

</ParamField>

## 请求示例

```bash
curl -X DELETE "https://api.tennda.ai/v1/assets/asset-20260508120001-xyz" \
  -H "Authorization: Bearer sk-xxxxxxxxxx"
```
