---
title: '删除素材'
description: 'DELETE /v1/assets/{virtual_id}'
api: 'DELETE https://api.gravitex.ai/v1/assets/{virtual_id}'
---

## 简介

删除指定素材（上游 + 本地映射）。

## 认证

<ParamField header="Authorization" type="string" required>
  Bearer Token，如 `Bearer sk-xxxxxxxxxx`
</ParamField>

## 路径参数

<ParamField path="virtual_id" type="string" required>
  素材虚拟 ID
</ParamField>

## 请求示例

```bash
curl -X DELETE "https://api.gravitex.ai/v1/assets/asset-20260508120145-pqwhc" \
  -H "Authorization: Bearer sk-xxxxxxxxxx"
```

## 响应示例

```json
{
  "deleted": true,
  "virtual_id": "asset-20260508120145-pqwhc"
}
```
