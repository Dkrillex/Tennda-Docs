## 简介

Seedance 2.0（`seedance-2-0` / `seedance-2-0-fast`）支持文生视频、图生视频、多模态参考与素材库（`asset://`）。本文档为系列概览；各接口说明见下方子页面。

<Note>
  `asset://` 须按素材 `asset_type` 映射到 `image_url` / `video_url` / `audio_url`。素材仅接受公网 **https URL** 创建，不支持 Base64 直传。
</Note>

## 认证

所有接口使用 Bearer Token：`Authorization: Bearer sk-xxxxxxxxxx`

## 模型

| 模型 ID | 说明 |
|---------|------|
| `seedance-2-0` | 标准版，支持 480p / 720p / 1080p |
| `seedance-2-0-fast` | 快速版，支持 480p / 720p |

## 推荐流程

1. （可选）创建素材组 → 创建素材 → 轮询至 `status: active`  
2. `POST /v1/video/generations` 提交任务  
3. `GET /v1/video/generations/{task_id}` 轮询至 `completed`  
4. 若失败且需排查审核，调用 [查询审核拦截原因](/cn/api-reference/endpoint/seedance-2.0/get-moderation-result)

统一视频入口说明亦见 [提交视频任务](/cn/api-reference/endpoint/submit-video-task)。
