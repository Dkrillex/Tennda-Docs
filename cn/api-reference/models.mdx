## 一个接口,接入 100+ 模型

**TenndaAI** 是面向 Agent 场景的统一 AI 模型接入平台。一套兼容 **OpenAI SDK** 的接口,即可路由到 100+ 主流模型,无需为每家厂商重复对接、无需订阅、按量计费。

<Note>
**Better prices, higher uptime, no subscriptions.** — 更优价格、更高可用、无需订阅。分布式基础设施支持自动故障转移,某家供应商不可用时自动切换备份;支持组织级数据策略,精细控制哪些供应商可访问数据。
</Note>

## 💰 定价说明

### 计费方式

- **按量计费**:根据实际使用的 Token 数量计费
- **无最低消费**:充多少用多少,余额永不过期
- **实时扣费**:每次调用后立即从余额扣除费用

### 价格优势

- 官方源头转发,价格较直连具备优势
- 批量使用可联系商务获取专属价格

### 成本优化建议

1. **分级使用**:简单任务用便宜模型,复杂任务用旗舰模型
2. **测试优化**:先用小模型测试,确定需求后再用大模型
3. **批量处理**:大量相似任务可以选择 Mini / Lite 版本
4. **缓存复用**:对重复查询结果进行缓存
