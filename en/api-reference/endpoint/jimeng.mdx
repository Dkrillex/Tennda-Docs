## Introduction

Jimeng video uses `POST /jimeng/` with query parameter `Action`:

- **Submit**: `Action=CVSync2AsyncSubmitTask`
- **Get result**: `Action=CVSync2AsyncGetResult`

Also pass required `Version`.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer Token, e.g. `Bearer sk-xxxxxxxxxx`
</ParamField>

## Examples

<Tabs>
  <Tab title="Submit">
    ```bash
    curl -X POST "https://api.tennda.ai/jimeng/?Action=CVSync2AsyncSubmitTask&Version=2024-06-01" \
      -H "Authorization: Bearer sk-xxxxxxxxxx" \
      -H "Content-Type: application/json" \
      -d '{
        "req_key": "jimeng_video_generation",
        "prompt": "A cat playing piano"
      }'
    ```
  </Tab>
  <Tab title="Get result">
    ```bash
    curl -X POST "https://api.tennda.ai/jimeng/?Action=CVSync2AsyncGetResult&Version=2024-06-01" \
      -H "Authorization: Bearer sk-xxxxxxxxxx" \
      -H "Content-Type: application/json" \
      -d '{
        "req_key": "jimeng_video_generation",
        "task_id": "async_task_id_from_submit"
      }'
    ```
  </Tab>
</Tabs>

For other models, use [Submit video task](/en/api-reference/endpoint/submit-video-task) (`/v1/video/generations`).
