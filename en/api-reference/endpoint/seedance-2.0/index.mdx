## Introduction

Seedance 2.0 (`seedance-2-0` / `seedance-2-0-fast`) supports text-to-video, image-to-video, multi-modal references, and an asset library (`asset://`). This page is the series overview; each endpoint has its own page below.

<Note>
  Map each `asset://` by `asset_type` to `image_url`, `video_url`, or `audio_url`. Assets must be created from public **https** URLs only (no Base64 upload).
</Note>

## Authentication

All endpoints: `Authorization: Bearer sk-xxxxxxxxxx`

## Models

| Model ID | Description |
|----------|-------------|
| `seedance-2-0` | Standard; `480p` / `720p` / `1080p` |
| `seedance-2-0-fast` | Fast; `480p` / `720p` (no 1080p) |

## Recommended flow

1. Optional: create asset group → create asset → poll until `status: active`  
2. `POST /v1/video/generations`  
3. `GET /v1/video/generations/{task_id}` until `completed`  
4. On moderation failure, see [Get moderation result](/en/api-reference/endpoint/seedance-2.0/get-moderation-result)

See also [Submit video task](/en/api-reference/endpoint/submit-video-task).
