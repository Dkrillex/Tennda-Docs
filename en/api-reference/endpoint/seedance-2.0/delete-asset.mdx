---

## Introduction

Delete one asset (upstream + local mapping).

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer Token, e.g. `Bearer sk-xxxxxxxxxx`
</ParamField>

## Path parameters

<ParamField path="virtual_id" type="string" required>
  Virtual asset ID
</ParamField>

## Example

```bash
curl -X DELETE "https://api.tennda.ai/v1/assets/asset-20260508120145-pqwhc" \
  -H "Authorization: Bearer sk-xxxxxxxxxx"
```

## Response

```json
{
  "deleted": true,
  "virtual_id": "asset-20260508120145-pqwhc"
}
```
